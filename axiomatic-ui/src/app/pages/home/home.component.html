AI attestations → PoVal™ on-chain proofs. Submit an ACJ-1 canonical JSON describing the asset; the oracle validates and normalizes it, runs the trained model, and returns a PoVal (Proof of Valuation): a signed valuation with uncertainty and an explainable factor breakdown, cryptographically bound to inputs, model manifest, and code hash. With one click we anchor a tamper-proof hash on Algorand (low fees, fast finality) and register it in the Axiomatic Attestation Registry, returning a txid, a verification link, and a registry ID. Metadata follows ARC-3/19 and can optionally be packaged as a W3C Verifiable Credential—auditable by anyone. Proofs, not data. No PoVal, no asset.
